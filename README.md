## NOTE
This project is a personal research project aimed at better understanding the security and client-side behavior of the SOYO mobile application.
It has no malicious intent.

## Scope
The goal of this project is to:
- Understand mobile application network behavior
- Analyze API communication patterns
- Identify potential security and business-logic risks
- Improve reverse engineering and dynamic analysis skills

## What was looked at
- Login and session handling
- Some publicly reachable API endpoints
- Request formats used by the mobile client
- Runtime behavior during normal app usage

> ⚠️ Sensitive data such as tokens, endpoints, payloads and scripts have been
> **redacted or simplified** to prevent misuse.

## Ethical Notice
This project is for learning and research purposes only.
No real user data is shared and everything was tested on my own accounts.

## Tools & Topics
- Android app analysis
- Network traffic inspection
- Runtime instrumentation concepts
- API behavior observation

## Disclaimer
This repository is intended for educational and security research purposes only.
